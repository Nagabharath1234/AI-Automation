Generating Text to Image and Video - n8n Workflow Documentation
Overview
This n8n workflow is a comprehensive AI-powered video generation system that creates videos from text prompts and images using multiple AI models and services. The scenario demonstrates five different video generation approaches: basic text-to-video conversion, image-to-video animation, "Cameos" (personalized video creation), storyboard-based video generation, and AI-enhanced text-to-video with Google Gemini integration. The workflow is manually triggered and uses the kie.ai API (which provides access to advanced AI video models including Sora 2 and Runway) to generate professional-quality videos from various inputs. Each generation path includes polling mechanisms to wait for video processing completion before retrieving the final results.
Connected Applications

kie.ai API (AI video generation service - Sora 2, Runway models)
Google Gemini AI (for advanced prompt processing)
Image hosting service (image2url.com for source images)

Workflow Execution Flows
Flow 1 - Basic Text to Video:
Manual Trigger → Video Prompt → Text to video → Wait → Get Video → If1 → Video URL (or loop back to Wait)
Flow 2 - Image to Video:
Manual Trigger → Image url → Video prompt 1 → Image to Video → Wait1 → Get Video 1 → If 2 → Video URL 1 (or loop back to Wait1)
Flow 3 - Cameos:
Video Prompt 2 → Text to Video1 → Wait2 → Get Video1 → If → Video URL1 (or loop back to Wait2)
Flow 4 - Storyboard:
Video Prompts → Story Request → Wait3 → Get Video 2 → If2 → Video URL 2 → Default (or loop back to Wait3)
Flow 5 - AI-Enhanced Text to Video:
Video prompt 3 → AI Agent (powered by Google Gemini Chat Model) → Text to Video → Wait4 → Get Video 4 → If 5 → Video URL 4 (or loop back to Wait4)
Auxiliary Flow:
Bad → Good
